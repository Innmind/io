# Changelog

## 2.2.0 - 2023-10-15

### Added

- `Innmind\IO\Readable\Stream::size()`

## 2.1.0 - 2023-10-15

### Added

- `Innmind\IO\Readable\Chunks::lazy()`

- Support for `innmind/immutable:~5.0`

- All encodings are represented with `Innmind\Immutable\Str\Encoding`
